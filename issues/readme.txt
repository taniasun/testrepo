Prerequisites: Java 8, Maven

Run from cmd (windows)
1. unzip to [folder]
2. add extention .exe to chromedriver file \issues\drivers (gmail does not allow .exe in attachments)
3. in cmd change dir to [folder]
4. run mvn install


From eclipse IDE:
1. unzip to [folder]
2. in eclipse select Import -> Existing mvn project
3. select [folder] -> finish
4. right click on test.java -> run as -> jUnit test
